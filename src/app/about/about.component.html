Bird info by Dan Miller

This is an Internet of Things monitoring solution, running on a beaglebone. Using Angular 2 with Bootstrap on the frontend, and Express, RabbitMQ, and MySQL on the backend.

Angular 2

Angular 2 is the front-end framework used to display information about and interact with the IoT devices. Chosen to give me a chance to try to figure it out.

Node

NodeJS with Express is used on the backend to serve up web pages and moderate interactions between the IoT devices and the database. Chosen for its enormous support and user base, making it fast to integrate with other frameworks.

BeagleBone

A BeagleBone Black is used to host the server, as a cheap and effective solution for a low-capacity network

RabbitMQ

RabbitMQ is used to manage lightweight communication between the IoT devices and the server or other devices. The open-source support for MQTT and Websockets were the main attractions of this framework.

MySQL

MySQL to manage persistent data storage. Chosen for familiar SQL interface and easy configuraton.
